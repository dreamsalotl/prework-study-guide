CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS data is linked by putting a stylesheet link in the header.
  • Class selectors affect every element with the specified class name.
  • You can also create specific class names to affect either one or multiple elements down the line.

Git

  • Git commands are directly usable from the terminal.

    • Notable Git commands:

  • git status: Checks what branch you are currently on.
  • git checkout -b branch-name: Creates a new branch and switches to it.
  • git add -A: Commit all files in the working branch.
  • git commit -m "Insert message here.": Associates a message with our commit to summarize implemented changes. Remember -m means "message".
